# Code Quality Rubric

## 1. Logic (1-10)

- **1**: Broken, doesn't work.
- **10**: Elegant, handles all edge cases, optimized.

## 2. Style (1-10)

- **1**: Unreadable, inconsistent.
- **10**: Idiomatic, perfect naming, self-documenting.

## 3. Testability (1-10)

- **1**: No tests, impossible to test.
- **10**: High coverage, modular, easy to mock.

## Overall Pass

Score >= 7 in all categories.
