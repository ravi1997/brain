# Check Index

## Purpose

A central point for all binary validation lists and scoring rubrics.

## Checklists (`checklists/`)

- `pr_review_checklist.md`: Manual steps for a reviewer.
- `deployment_checklist.md`: Steps before pushing to prod.
- `onboarding_checklist.md`: Steps when a new developer joins.

## Rubrics (`rubrics/`)

- `code_quality_rubric.md`: Scoring 1-10 on logic, style, and testing.
- `security_rubric.md`: Scoring 1-10 on safety and secrets.
