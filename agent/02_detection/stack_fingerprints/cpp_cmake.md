# Stack Fingerprint: C++ CMake

## Purpose

Identifies and configures for a C++ project using CMake.

## Signals

- `CMakeLists.txt`.
- `.cpp`, `.hpp`, `.cc` files.

## Configuration

- **Build**: `mkdir build && cd build && cmake .. && make`.
- **Test**: `ctest` or GTest integration.
- **Standard**: C++17 or C++20.
