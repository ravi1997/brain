# Stack Fingerprint: React Vite

## Purpose

Identifies and configures for a modern React frontend using Vite.

## Signals

- `vite` dependency.
- `vite.config.ts` or `vite.config.js`.
- `src/App.tsx` or `src/App.jsx`.

## Configuration

- **Build**: `npm run build`.
- **Dev**: `npm run dev`.
- **Framework**: React 18+.
- **Router**: Often `react-router-dom`.
