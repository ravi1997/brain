# Plan Output Contract: Naming Conventions

## Standard

`[PHASE_ID]_[DESCRIPTIVE_NAME].md`

## Examples

- `SRS_FUNCTIONAL_REQS.md`
- `ARCH_ENTITY_RELATIONSHIP.mermaid`
- `TEST_COVERAGE_REPORT.md`
- `REL_NOTES_V1.0.0.md`

## Rules

- Use uppercase for logical categories (SRS, ARCH, TEST).
- Use underscores instead of spaces.
- Always include the file extension (`.md`, `.mermaid`, `.json`).
