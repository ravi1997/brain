# Vulnerability Response Playbook

## Purpose

Action plan for when a security flaw is discovered.

## Steps

1. **Identify**: Confirm the vulnerability and its impact.
2. **Contain**: Temporarily disable the affected feature or limit access.
3. **Remediate**: Patch the code or update the dependency.
4. **Verify**: Run `08_security_review.md` to ensure the fix works.
5. **Report**: Document the incident in `plans/Security/INCIDENT_LOG.md`.
