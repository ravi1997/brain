## 2. Threat Identification (STRIDE)

- **Spoofing**: Can someone pretend to be another user?
- **Tampering**: Can data be modified in transit?
- **Repudiation**: Can a user deny an action?
- **Information Disclosure**: Can secrets be leaked?
- **Denial of Service**: Can the system be overwhelmed?
- **Elevation of Privilege**: Can a user get admin access?
