# Stack Gate: Python

## Rules

1. **Linting**: Must pass `pylint` or `flake8` with zero errors.
2. **Formatting**: Code must be formatted with `black`.
3. **Types**: Use type hints for all function signatures where possible.
4. **Environment**: `requirements.txt` or `pyproject.toml` must be up to date.

## Remediation

- Run `black .` and `isort .` before re-submitting.
