# Gate: Global Quality

## Pass Criteria

- [ ] 100% of unit tests pass.
- [ ] No Linter errors in new code.
- [ ] Build completes without warnings.
- [ ] Test coverage is >= 80% (or the project's current baseline).

## Fail Criteria

- [ ] Any test failure.
- [ ] Any linting error that breaks build.
- [ ] Presence of `TODO` or `FIXME` in the main logic path.

## Remediation

1. Re-read the failed log.
2. Adopt `profile_implementer.md`.
3. Fix the specific violation.
4. Re-run tests.
