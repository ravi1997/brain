# Skill Index

## Purpose

A registry of specialized tool-usage patterns and capabilities.

## Skills Folders

- `planning/`: Roadmap, Gantt, and Backlog generation.
- `analysis/`: Stakeholder interviews and Requirement extraction.
- `architecture/`: Mermaid generation and system modeling.
- `implementation/`: Refactoring, debugging, and code generation.
- `testing/`: Unit test generation and edge-case discovery.
- `security/`: Secret scraping and CVE analysis.
- `devops/`: Docker/CI setup and deployment scripts.
- `knowledge_extraction/`: Reading large codebases and summarizing.

## Skill Structure

Each skill file includes:

- **Inputs**: Data required.
- **Procedure**: Logic flow.
- **Failure Mode**: How to handle skill failure.
