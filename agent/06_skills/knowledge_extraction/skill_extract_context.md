# Skill: Knowledge Extraction

## Purpose

Efficiently reading and understanding a repository.

## Procedure

1. **Directory Map**: Run `ls -R` and identify core folders.
2. **Metadata Scan**: Read `package.json`, `README.md`, `.env.example`.
3. **Entry Search**: Find `index.js`, `main.py`, or `app.tsx`.
4. **Symbol Extraction**: Grep for Class and Function definitions.
5. **Relationship Mapping**: See which files import which.

## Outputs

- `plans/Architecture/CODEBASE_SUMMARY.md`.
