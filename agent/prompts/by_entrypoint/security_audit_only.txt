- `plans/Security/VULNERABILITIES.md` (if any found)
- `agent/09_state/SECURITY_STATE.md`

STOP CONDITIONS
- All scans complete
- Security gate evaluated
- Remediation plan created for any findings

FINAL RESPONSE
Report: Vulnerability count by severity, secrets found (masked), gate status, remediation steps.
